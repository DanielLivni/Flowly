# בונה זרימה

עורך ויזואלי בעברית וב-RTL לבניית עצי החלטה ותסריטי שירות, כולל עריכת שלבים, חיבורים, פרטי תסריט, ייבוא YAML וייצוא YAML.

## הרצה מקומית

```bash
npm install
npm run dev
```

## build לפרודקשן

```bash
npm run build
```

## מבנה הקוד

סקירה מלאה של המודל, קבצי הקוד המרכזיים, סכמת ה-YAML והאחריות של כל מודול נמצאת ב-[CODE_OVERVIEW.md](./CODE_OVERVIEW.md).
